# Intelek Compute Examples

Coleccion de funciones y scripts de prueba para cubrir combinaciones de entradas y salidas.

Estructura:
- `functions/`: modulo con 12 funciones (todas las combinaciones de 0-3 entradas y 0-2 salidas).
- `scripts/python/`: programas simples de Python.
- `scripts/notebooks/`: notebooks de Jupyter de demostracion.

Requisitos:
- `pandas` solo es necesario para la funcion que devuelve DataFrame.

Ejemplo rapido:
```bash
python scripts/python/run_all_functions.py
```
